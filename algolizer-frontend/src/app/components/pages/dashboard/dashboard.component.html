About this application


- Primary purpose of this application is visual revision and/or learning of sorting algorithms. It is intended for all students of technical faculties, as well as algorithms enthusiasts who would like to know a thing or two about sorting algorithms.

- 'Show statistics' option shows you summary information about all exercises that were done on this website, by all the users. This summary includes number of exercises, average points per one exercise, average points per algorithm, and other statistic info.

- Application usage: firstly, you need to choose an algorithm you want to learn or revise from dropdown menu. After clicking the 'Start visualisation!' button, you will be prompted to interactive visualisation of the algorithm. When you finish the visualisation part, you will be prompted to do a short exercise that will test your knowledge of the algorithm. Array which is used for visualisation, as well as within excercise, is randomly generated. As a result of the excercise, you will see your score. Your score will update overall database of statistics, that will then be used to find the thoughest algorithms, ie. the algorithms that you see on dashboard. This info is obtained from a complex recommendation-based system, which operates based on statistics data.
